Інструкція для додавання платіжної системи на OpenCart

Як встановити та підключити платіжний сервіс для CMS OpenCart?
  1. Зареєструватися або увійти в особистий кабінет.
  2. Перейти за посиланням «Плагіни для CMS» та завантажити платіжний модуль для OpenCart.
  3. Встановити модуль на вашому сайті.
  4. Налаштувати всі способи оплати, використовуючи параметри вашого магазину.

Після цього оплата платіжними картками та електронними гаманцями буде доступна на вашому сайті!
